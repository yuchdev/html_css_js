Tables

Table basics

The <table> element is used to create a table. The contents of the table are written out row by row.
To indicate the start of each row using the opening <tr> tag. The <tr> stands for table row.
Each cell of a table is represented using a <td> element. The <td> stands for table data.
The <th> element defines a table header.
A more complex HTML table may also include <caption>, <col>, <colgroup>, <thead>, <tfoot>, and <tbody> elements.
There are a number of table attributes, like align, bgcolor, border, cellpadding, cellspacing, frame, rules, summary, width. These attributes are obsolete and not supported in HTML5.
Use CSS styling instead

Spanning cells

Sometimes you may need the entries in a table to stretch across more than one column or row. The colspan or rowspan attribute can be used on a <th> or <td> element and indicates how many columns that cell should run across.

Table structure

There are three elements that help distinguish between the main content of the table and the first and last rows (which can contain different content) and style these sections in a different manner than the rest of the table (using CSS rules)

<thead> is a headings of the table, The body should sit inside the <tbody> element, The footer belongs inside the <tfoot> element.

Table columns

The <col> tag specifies column properties for each column within a <colgroup> element, instead of repeating the styles for each cell. The attribute span specifies the number of columns the <col> element should span.

Table styling

There are some outdated attributes which you should not use on new websites. You may, however, come across some of them when looking at older code, so I will mention them here. All of these attributes have been replaced by the use of CSS. For example, you could set style attributes like width, cellpadding, cellspacing, border and background color etc.
