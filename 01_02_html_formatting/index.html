Demonstration of formatting tags

Structural markup

Text formatting

Structural markup changes visual appearance of the text.
Tag <b>: Bold text
Tag <i>: Italic text
Superscript example, <sup> tag: 4th September
Subscript example, <sub> tag: The amount of CO2 in the atmosphere grew by 2ppm in 2009
Use the <mark> tag if you want to highlight parts of your text
The <small> tag defines smaller text (and other side comments)
The <u> tag represents some text that should be stylistically different from normal text, such as misspelled words

Date and time

The <time> tag defines a human-readable date/time (see source for examples):
We open at 10:00 every morning
I have a date on Valentines day

Insert and delete

The <ins> element can be used to show content that has been inserted into a document

The <del> element can show text that has been deleted from it

The content of a <ins> element is usually underlined, while the content of a <del> element usually has a line through it.

The <s> element indicates something that is no longer accurate or relevant (but that should not be deleted). Visually the content of an <s> element will usually be displayed with a line through the center. Example:
Laptop computer:
Was $995
Now only $375

Text direction

Use <bdo> tag, right-to-left text example (notice, by symbols, not words):
This text will go right-to-left
This text is explicitly left-to-right, even though also uses <bdo>, which is default

This text contains <bdi> tag; it shows the browser, that this content may have different direction, for example Arabic word in English text: إيان

Pre-formatted text

Text in a <pre> element is displayed in a fixed-width font (usually Courier), and it preserves both spaces and line breaks: 

        Text in a pre element is displayed in a fixed-width font (usually Courier), 
        and it preserves both spaces and line breaks.    (4 spaces)

Code example

Display meter or progress bar

The <meter> tag defines a scalar measurement within a known range, or a fractional value.
Examples: Disk usage, the relevance of a query result, etc.

Display a meter:


2 out of 10 <meter value="2" min="0" max="10">2 out of 10</meter>
60% <meter value="0.6">60%</meter>

Note! The <meter> tag should not be used to indicate dynamic progress (as in a progress bar). For progress bars, use the <progress> tag:
<progress value="22" max="100"></progress>

Semantic markup

<strong> and <em>

The use of the <strong> tag: element indicates that its content has strong importance. For example, the words contained in this element might be said with strong emphasis. By default, browsers will show the contents of a strong element in bold.
The use of the <em> tag: the element indicates emphasis that subtly changes the meaning of a sentence. By default browsers will show the contents of an <em> element in italic.

Blockquote and quotations

The <blockquote> element is used for longer quotes that take up an entire paragraph. Note how the <p> element is still used inside the <blockquote> element.

Browsers tend to indent the contents of the <blockquote> element, however you should not use this element just to indent a piece of text, rather you should achieve this effect using CSS.

HTML and CSS, Jon Ducket

The <q> element is used for shorter quotes that sit within a paragraph. Browsers are supposed to put quotes around the <q> element, however Internet Explorer does not — therefore many people avoid using the <q> element. Both elements may use the cite attribute to indicate where the quote is from (HTML and CSS, Jon Ducket) Its value should be a URL that will have more information about the source of the quotation.

Abbreviations, acronyms and references

When you are referencing a piece of work such as a book, flm or research paper, the <cite> element can be used to indicate where the citation is from. Example:

A Brief History of Time by Stephen Hawking has sold over ten million copies worldwide

The first time you explain some new terminology (perhaps an academic concept or some jargon) in a document, it is known as the defining instance of it. The <dfn> element is used to indicate the defining instance of a new term. Some browsers show the content of the <dfn> element in italics. Safari and Chrome do not change its appearance.

Use of <abbr title="Professor"> tag: Prof. Stephen Hawking is a theoretical physicist and cosmologist.
If you use an abbreviation or an acronym, then the element can be used. A title attribute on the opening tag is used to specify the full term

The <address> element has quite a specific use: to contain contact details for the author of the page.
It can contain a physical address, but it does not have to. For example, it may also contain a phone number or email address. Browsers often display the content of the <address> element in italics. Example:

homer@example.org

742 Evergreen Terrace, Springfield.
