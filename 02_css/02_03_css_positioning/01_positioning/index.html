CSS Positioning Properties

Types of CSS position property

The position property is used to set the positioning method of an element. It is easily to understand by looking at the examples below, rather than explanation. CSS offers the following types of positioning:

  • static - default value, or normal document flow
  • relative - element is positioned relative to its normal position
  • fixed - element is positioned relative to the document itself
  • absolute - element is positioned relative to the nearest positioned ancestor
  • sticky - element is positioned based on user's scroll position

Sticky

This is sticky positioning, or positioned based on user's scroll position. The element is not affected by the scroll position of the document, and always remains in the same position. Sticky positioned element is removed from the normal flow of the document.
Note: Safari requires the -webkit-sticky property for the element to be sticky. Obviously, in order to make the element sticky, at least one of the following properties must be set: top, bottom, left, or right (we must set where elements should be stuck to).
