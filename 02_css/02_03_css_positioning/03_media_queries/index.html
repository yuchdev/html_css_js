CSS Media Queries

Definition

CSS Media Queries are used to specify the behavior of a conditional style sheet, depending on screen size, orientation, device. E.g. screen, print, tv, projection, handheld, braille, aural, all. Media Queries is a basis of a Responsive Web Design (RWD) approach.

Responsive Design

Responsive Design is a technique that allows web pages to be designed to respond to the properties of the screen. It's a way to provide better user experience for different screen sizes, portable and handheld devices. In the beginning of mobile development, mobile version of a website were served as a different URL, that led to errors and running different versions of the website on different devices. Meanwhile, responsive design allows you to create a single code base that fits all devices.
