CSS Flexbox

Flex container

CSS Flexbox is the more modern way to create a responsive layout. You should create a flexible container, and then you can add flex items to it (CSS property display: flex). The flex items will be automatically float as if you use float property. The direct children of the flex container will be flex items.

Flex justify

You can change the alignment of the flex items using justify-content property. It aligns items across the main axis. The default value is flex-start, which is floating the items to the left.

Possible values of justify-content property:

  • flex-start - aligns items to the left side of the container
  • flex-end - aligns items to the right side of the container
  • center - aligns items at the center of the container
  • space-between - displays items with equal spacing between them
  • space-around - displays items with equal spacing around them
  • start - aligns items to the left side of the container (same as flex-start)
  • end - aligns items to the right side of the container (same as flex-end)
  • normal - sets this property to its default value
  • stretch - stretches items to fit the container
  • space-between - displays items with equal spacing between them
  • space-around - displays items with half-size spacing around them
  • space-evenly - displays items with equal spacing around them

Another aligning property is align-items. Unlike justify-content, it aligns items across the cross axis. Possible values are:

  • flex-start - aligns items to the top of the container
  • flex-end - aligns items to the bottom of the container
  • center - aligns items at the vertical center of the container
  • baseline - aligns items at their baselines
  • stretch - stretches items to fit the container

Perfect centering

Flex containers are perfect for placing elements in the center of the page. We just use align-items: center and justify-content: center.
