Responsive Web Design

How RWD works

In the early days of the internet, websites were mostly viewed on desktop computers, thus, only one version of a website was needed, with static size and design.
However, with the rise of mobile devices, the need for a single version of a website that works well on all devices became apparent.

Viewport

The viewport is the user's visible area of a web page. It varies with the device - it will be smaller on a mobile phone than on a computer screen.
In order to let the browser know we utilize Responsive Design, we need to add the following meta-tag to the head section of our HTML document:
<meta name="viewport" content="width=device-width, initial-scale=1.0">

  • width=device-width sets the width of the page to follow the screen-width of the device
  • initial-scale=1.0 sets the initial zoom level when the page is loaded

Fitting the content to the screen

There are simple rules to follow in order to make your website responsive:

  • The users got used to vertical, but not horizontal scrolling. Horizontal scrolling considered a bad user experience
  • Use relative units of measurement, such as percentages, ems, rems, etc.
  • Do not design particularly large (wide) elements, as they will not fit on smaller screens
  • Always start your design with the smallest screen size for mobile devices

Grid View

Grid view is a layout that consists of a number of columns and rows, which are used to fit the content, that may take one or more columns and/or rows.

Grid View multi-column layout example

Here we try to provide different number of columns for different screen sizes. In our particular example, we use 8 columns maximum, so we calculate the width of each column as follows: 100% / 8 = 12.5%
On practice, modern CSS frameworks, like Twitter Bootstrap, support up to 12 columns.

Using Media Queries, we can provide different number of columns for different screen sizes, e.g. for
@media screen and (min-width: 950px) and (max-width: 1199px)
we provide 4 columns, so the width of each column is 25%.
