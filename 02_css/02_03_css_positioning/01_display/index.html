CSS display Property

Definition

The display property specifies if and how HTML element is displayed, and it is one of the most important CSS properties.

In the beginning of HTML chapter we were talking of block and inline elements. These properties are given by some meaningful default to certain elements, for example, <p>, <div>, <header>, <footer> are block elements, and <span>, <a>, <img> are inline elements.

As you see, this paragraph is block element, i.e. it takes all the available width. This span is inline element, i.e. it is displayed within the line.

Changing display property

However, these fundamental properties could be changed by display property. Let's say, list elements <li> are displayed as block elements, but we can change this property to inline elements.

Using display property to make elements hide and appear

Sometimes we need to hide elements, and sometimes we need to show them. For this purpose we can use display: none property.

It should not be confused with visibility: hidden property; while display: none property removes element from the document flow, visibility: hidden property just hides the element
