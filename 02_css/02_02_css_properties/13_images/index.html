CSS Styling Images

Styling Images

Round Corners

The border-radius property allows you to round the corners of an element.

Thumbnail Images

Ise the border property to add a border around an image.

Image Scaling

Use max-width and max-height properties to scale an image to fit the container, but never make it larger than its original size.

Transparent Design

Use the opacity property to make an image transparent.

Image Reflection

Use the box-reflect property to reflect an image. The property is not fully supported by all browsers, so you can use engine-specific prefixes:
-webkit-box-reflect: below;

Image Sprites

A web page with many images can take longer time to load, as it generates multiple server requests. An image sprite is a collection of images put into a single image, which shortens loading time.

Image Gallery

CSS can be used to create a responsive image gallery
