CSS Colors

Color specifications

Color in HTML5 can be specified one of the following ways

  • by using the # symbol and a hexadecimal number
  • by using the rgb() function
  • by using the rgba() function
  • by using the hsl() function
  • by using the hsla() function
  • by using the currentColor keyword
  • by using the transparent keyword
  • by using the inherit keyword

RGB defines intensity of red, green and blue components of a color. The values of the components are in the range from 0 to 255.
Example: rgb(255, 0, 0)

RGBA defines intensity of red, green, blue and alpha components of a color. The values of the components are in the range from 0 to 255.
Example: rgba(255, 0, 0, 0.5)

You can define transparency separately using the opacity CSS property.
Example: background-color: green; opacity: 0.5

HSL defines hue, saturation and lightness of a color. The values of the components are in the range from 0 to 360, 0 to 1 and 0 to 1.
Example: hsl(0, 30%, 70%)

HSLA defines hue, saturation, lightness and alpha components of a color. The values of the components are in the range from 0 to 360, 0 to 1 and 0 to 1.
Example: hsla(50, 30%, 70%, 0.5)

The currentColor keyword is used to specify the color of the element that the color property is applied to (select the next line to read the text).
Example: color: currentColor;

The transparent keyword is used to specify a transparent color (select the next line to read the text).
Example: color: transparent;

The inherit keyword is used to specify the color of the parent element, from paragraph in this case.
Example: color: inherit;

Elements to color

You can color text, background, border, and outline of any element.

Background: blue, text: orange, border: tomato, outline: magenta

background: #7fecff, text: #5f3aff, border: #ff2631, outline: #16ff36

You can add transparency to any color by using the rgba() function.

background: rgba(255, 0, 0, 0.3), text: rgba(0, 255, 0, 0.7), border: rgba(0, 0, 255, 0.0), outline: rgba(255, 255, 0, 0.5)

Gradient

Gradient is a smooth transition between two or more colors, and you can use the linear-gradient() function with the background property to create it. There are 3 types of gradients: linear, radial and conic.
