CSS styling of text in HTML document

Font family

CSS has lots of properties for formatting text. Choosing the right font has a critical importance for readability and user experience, the fond adds value to the document.

You can style font family of text in HTML document by using font-family CSS property. Usually several font families are mentioned, separated by commas. It happens because different browsers may have different sets of fonts, and the user may simply not be having the particular font. Therefore, it is very important to always use fallback fonts

However, some fonts are "safer" than others, and most likely present in every browser:

  • Arial
  • Times New Roman
  • Courier New
  • Helvetica
  • Verdana
  • Georgia

There are 5 general families of fonts:

  • Serif has small strokes at the edges. This is strong and formal font
  • Sans-serif has straight lines and has no strokes. This is a good choice for casual writing, creating modern look
  • Monospace is a font with fixed width, this looks kind of "mechanic" and could be a good choice for code
  • Cursive is a font with curved strokes, this is a good choice for old-fashioned writing, imitating human handwriting
  • Fantasy is a font with a lot of decorative strokes

Color

Text color could be specified with the name of the color, or with the hexadecimal code. Example:

color: red;
color: #ff0000;

Size

Text size could be specified with the font-size CSS property. It could be measured absolutely or relatively. Absolute size is measured in pixels or points.

Relative text size is measured in percentage or measure units called ems. Important: When you mention the same element in several rules, relative size is not being overwritten, but rather it is accumulated (e.g. 120% * 2em = 240%). It is not recommended to mix different measure units in one document.

This is 120% sized text

This is 2em sized text

The text size can be set with a vw unit, which means the viewport width. Such text will follow the size of the browser window upon resizing.
Note: Viewport is the browser window size. 1vw = 1% of viewport width. If the viewport is 50cm wide, 1vw is 0.5cm.

Transformation and alignment

CSS property text-transform could be used to change the case of the text (uppercase, lowercase, capitalize).

CSS property text-align could be used to align the text (left, right, center, justify).
