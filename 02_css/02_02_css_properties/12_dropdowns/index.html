CSS Styling Dropdowns

Definition

Dropdowns in a context of CSS is a box with basically everything, that appears when user clicks or hovers over an element (this concept is much wider than just dropdown input element).

Tooltip

The simplest use of dropdowns is a tooltip. You hover over an element and some explanation text appears. Explaining an example:

  • You can use any element as a container for a tooltip, in our case we use div, and give it a class tooltip
  • Put inside the container the content you want to have a tooltip, and the tooltip content as well
  • Add a class tooltip-content to the content
  • On the CSS side create classes tooltip and tooltip-content
  • We want the tooltip to be placed right below the element, so we give position: relative to a tooltip, and position: absolute to a tooltip-content
  • The tooltip content is hidden by default (property display: none), we make it visible (property display: block) by hovering over the tooltip, using pseudo-class .tooltip:hover
  • Other properties of the tooltip content depend on the tooltip element, and general style of a page. For example, property box-shadow: 0px 8px 16px 0px rgba(0,0,0,0.2) instead of solid border, gives the content a temporary-box look

Dropdown Menu

This one is probably the most common use of dropdowns, not as simple as tooltip, however, being created on the same principles.

  • The class class="dropdown-menu" is the same as for the tooltip
  • We style the button with class="dropdown-menu-button" to give it a dropdown-menu look
  • Dropdown menu content is hidden by default (property display: none), and displayed when the button is hovered: .dropdown-menu:hover .dropdown-menu-content. We also slightly change the button itself
  • We give links in our menu a view of "menu items" (content of .dropdown-menu a section), and we change their style while hovering over them (using pseudo-class .dropdown-menu-content a:hover)
