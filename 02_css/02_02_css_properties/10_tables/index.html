Using pseudo-class or pseudo-element for advanced table styling

You can use pseudo-class or pseudo-element for advanced table styling. For example, you can add some interactivity by using :hover pseudo-class for styling the table when the mouse is over it, and :active pseudo-class when the mouse is pressed down on it. And you can use :nth-child for zebra-styled tables, use CSS property background-color with :nth-child(even) or :nth-child(odd).

Scrolling tables

A responsive table will display a horizontal scroll bar if the screen is too small to display the full content. For that you can use overflow-x: auto CSS property, not to table itself, but to its parent element, like div or section.
