Sizing elements with CSS

Setting elements size with CSS

Basics of setting elements size with CSS are very intuitive:

  • width property sets width of an element;
  • height property sets height of an element;
  • max-width property sets maximum width of an element;
  • max-height property sets maximum height of an element;
  • min-width property sets minimum width of an element;
  • min-height property sets minimum height of an element;

Element size could be measured in any unit, as we saw from CSS units chapter. For example, this paragraph has height 100px and width 70%.

By default, element size does not include padding, border and margin. As we see in the next chapter, this property could be changed with box-sizing CSS property.

Special values of size properties

Size-related CSS properties may have special values:

  • auto value sets size to its default value;
  • initial value sets size to its default value;
  • inherit value sets size to the size of the parent element;

Use of max and min sizing properties

For example, max-width and max-height properties set maximum size of an element. Maximum width helps in handling devices with small window width. If we firmly set maximum width of an element to 500px, it will remain like that even if display size of a browser is smaller. On the contrary, if we set maximum width of an element to 100%, it will be resized with the browser. To see the difference, try to resize your browser window width below 500px.
