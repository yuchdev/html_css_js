CSS Styling Lists

What you can style in lists

It seems, not that much, but you can style the following elements in lists:

  • Change shape and color of an unordered list bullets
  • Set any image as an unordered list bullets
  • Change background of the list items
  • Change format of numbers/letters in ordered lists
