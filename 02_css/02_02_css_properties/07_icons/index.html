CSS Styling of Icons

Icon Libraries

Icons could be easily added to the page by using any inline tag, e.g. <i> or <span> Name of the icon is placed in the class attribute of the HTML element.
E.g. <i class="fas fa-home"></i>

Icons are being downloaded by using icon libraries, such as

  • Font Awesome
  • Google Material Icons
  • Bootstrap Icons

Icons are usually connected in the <head> section of the page, and could be used anywhere further. However, there are slightly different methods

Connecting Font Awesome

Font Awesome has Free and Pro versions, that vary in the number and quality of icons. In any case, you should create an account to start using icons. You can create an account on Font Awesome and start using icons. Usage is pretty straightforward, in the header of the page you create a script element
<script src="https://kit.fontawesome.com/bf1ecf6a20.js" crossorigin="anonymous"></script>, where bf1ecf6a20.js is your unique hash, created for your account.

Connecting Google Material Icons

Google Material Icons is a set of free libraries of icons, that is used in the majority of the web. Unlike Font Awesome, it is connected with a link:
<link rel="stylesheet" href="https://fonts.googleapis.com/icon?family=Material+Icons"> , where family is the name of the icon library.
There are several libraries, such as Material Icons, Material Icons Outlined, Material Icons Round, Material Icons Sharp, Material Icons Two Tone, and Material Icons Wide.

If you'd like to combine them in your document, you should add the new family using the "|" symbol, e.g.
<link rel="stylesheet" href="https://fonts.googleapis.com/icon?family=Material+Icons|Material+Icons+Outlined">

Examples of Google Material Icons: home person person_outline person_add person_pin person_pin_circle person_outline person_add_disabled person_remove pets refresh restaurant restaurant_menu download settings settings_applications settings_backup_restore settings_bluetooth

Connecting Bootstrap Icons

Connecting Bootstrap glyphicons is being done the same way as with Google Material Icons, only you include a link to the Bootstrap stylesheet: <link rel="stylesheet" href="https://stackpath.bootstrapcdn.com/bootstrap/4.4.1/css/bootstrap.min.css" crossorigin="anonymous">

Styling icons

Styling Font Awesome

As previously mentioned, icons are being used by using the class attribute of the HTML element. First, icons have simple naming rules, making it easier to choose the right icon. Let's say, in class fas fa-clock letter code s stands for solid, and far fa-home letter code r stands for regular.

<i class="fas fa-clock">

<i class="far fa-home">

However, the most important here is the fact the icon inherits the style of the parent element, and it is possible to change style of the icon by changing the style of the HTML icon container. E.g.

<i class="fas fa-clock" style="color: red;">

<i class="fas fa-check-square" style="font-size:60px;color:#2196F3">

<i class="fas fa-check-square" style="font-size:60px;color:#2196F3;text-shadow: 0 0 10px rgba(0,0,0,0.5); border: solid magenta 2px">

<i class="fas fa-check-square" style="font-size:60px;color:#2196F3;text-shadow: 0 0 10px rgba(0,0,0,0.5);">

Classes for icon sizing

There are special classes for adjusting size of the icon for different screen types: fa-xs, fa-sm, fa-lg, fa-1x, fa-2x, fa-3x, fa-4x, fa-5x, fa-6x, fa-7x, fa-8x, fa-9x, fa-10x
E.g.

<i class="fas fa-check-square fa-lg">

<i class="fas fa-check-square fa-2x">

<i class="fas fa-check-square fa-3x">

Stacking icons

You can get completely new icon by stacking them one of another, if they fit together. For that, you should apply class fa-stack to the parent element, and then add class fa-stack-1x or fa-stack-2x to the icon.

Example: Do not make photos!
<span class="fa-stack fa-lg">
  <i class="fas fa-camera fa-stack-1x"></i>
  <i class="fas fa-ban fa-stack-2x text-danger" style="color:red;"></i>
</span>

Rotating icons

Animating icons

Icons support simple animation by adding class fa-spin or fa-pulse.

Example:
<i class="fas fa-sync-alt fa-spin"></i>
<i class="fas fa-sync-alt fa-pulse"></i>
<i class="fas fa-cog fa-spin"></i>
<i class="fas fa-cog fa-pulse"></i>
