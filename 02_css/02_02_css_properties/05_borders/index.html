Styling border, margin, padding and outline

CSS Box Model

In HTML every element is considered a box. In the center of the box it has a content. Outside the content comes padding, then border, and finally margin. When you set the size of the box, its actually size of the content, but the actual size of the box is the sum of the content size and the padding, border and margin.

It could be overwritten by a CSS rule box-sizing, which could be set to border-box or content-box, with the default value of content-box. However, border-box could be more comfortable to work with as more explicit.

By default, the HTML element tries to fit its size according to the content size. If you restrain the size of the element, oversize content will be spilled out. This behavior could be changed with the overflow property, which could be set to auto, which is default, hidden and even scroll to scroll over the content. While vertical scrolling could be useful, horizontal scrolling is considered bad practice. Also, having more than one scrolling element inherited from one another often frustrates the user.

Border styling

Border width

There's a number of properties to style borders. You can set the border width using border-width property. Border width can be measured in px, pt, cm, em, or using pre-defined values: thin, medium, thick.

Border color

Just like any other element, border color can be set with a number of ways: hex e.g. #ffaacc, rgb, hsl, named color, or currentColor.

Border styling

Different border styles could be set using border-style property:

  • solid
  • dashed
  • dotted
  • double
  • groove
  • ridge
  • inset
  • outset
