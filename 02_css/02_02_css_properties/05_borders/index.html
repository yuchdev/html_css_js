Styling border and outline

Borders CSS properties

Border styling

There's a number of properties to style borders. However, in order to style the border with any of properties, e.g. color, size, rounding - you should define border-style first! List of common border styles:

  • solid
  • dashed
  • dotted
  • double
  • groove
  • ridge
  • inset
  • outset

Solid border, 5px: border: solid 5px black;

Thin border: border: medium black;

Double border, medium: border: double medium blue;

Groove border: border: groove 5pt green;

Ridge border: border: ridge 5pt red;

Border width

You can set the border width using border-width property. Border width can be measured in px, pt, cm, em, or using pre-defined values: thin, medium, thick.

Border color

Just like any other element, border color can be set with a number of ways: hex e.g. #ffaacc, rgb, hsl, named color, or currentColor.

Combining properties

Borders even can have different size for each side: border-style: dashed; border-width: thin medium thick 5px;

...and different style for each side:
border-size: 3px;
border-color: red;
border-top-style: solid;
border-right-style: dashed;
border-bottom-style: dotted;
border-left-style: double;

The border-radius property is used to add rounded borders to an element. It can be set in px, pt, cm, em, or using pre-defined values, e.g. border-radius: 10px;

Border properties can be set with individual properties: border-width, border-style, border-color, or use shortener border property, e.g. border: 5px solid #ffaacc, or border-left: 6px solid red

Image Borders

With the CSS property border-image you can add an image to the border of an element. Requirements for such an image are slightly complicated: it is a 3x3 grid of images, where the middle image is the content, corner images are corners of the border, and the edge images are the edges of the border, which could be stretched, repeated, or transparent.

This, the property has three parts:

  • the image to use as a border, URL or location
  • How to slice the image
  • The middle part of image border can be repeated, stretched, or rounded
