Elements background styling

Background could be specified as a color, a gradient, a pattern, an image or a combination of these. Complete list of background properties:

  • background
  • background-attachment
  • background-color
  • background-image
  • background-position
  • background-repeat
  • background-size

Any background-* CSS property is being overwritten by background property (without dash).

Note 1: When using the opacity property, all children inherit the opacity.
Note 2: Try to pick combination of text and background colors that will be readable.
