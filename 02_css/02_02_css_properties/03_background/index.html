Elements background styling

Background could be specified as a color, a gradient, a pattern, an image or a combination of these. These are useful background properties:

  • background
  • background-attachment
  • background-color
  • background-image
  • background-position
  • background-repeat
  • background-size
  • opacity

Any background-* CSS property is being overwritten by background property (without dash).

Note 1: When using the opacity property, all children inherit the opacity.
Note 2: Try to pick combination of text and background colors that will be readable. The same comes to background images.

The background-attachment property specifies whether a background image will be fixed or scrolls with the rest of the page.

You can use multiple background images, stacking them on top of each other, with the use of background-image property.

Special values for background size

The background-size property specifies the size of the background images. It may have two special values: cover and contain.

  • cover - The background image is resized to cover the entire element, even if it has to stretch or crop the image
  • contain - The background image is resized to make sure the image is fully visible

Contain image

background-size: contain

Cover image

background-size: cover

Full-size website background

If you choose to design your website with a full-size background image, there are simple rules applied:

  • Use background-size: cover to make sure the image is fully visible
  • Use background-attachment: fixed to make sure the image is fixed and does not have a scrolling effect
  • Center the image using background-position property

Background shadow

You can apply one or more shadows to an element with the box-shadow property.

Simple shadow

In the simplest case, you just use horizontal and vertical shadows, e.g.

box-shadow: 10px 10px;

Shadow with specific color

box-shadow: 10px 10px 5px #888888;

Blurred shadow

Add the third numeric parameter, blur radius, to create a blurred shadow.

box-shadow: 10px 10px 5px #888888;
