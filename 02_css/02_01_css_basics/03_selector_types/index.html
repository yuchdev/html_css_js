Element, class and ID selectors

Element selector

The browser traverse HTML document, looking for elements matching the style described in CSS stylesheet. First type of selector is element selector, and it applies the style to all elements in the document. It could be headers (<h1>, <h2>), paragraphs (<p>), lists (<ul>, <ol>), tables (<tr>, <td>) and so on.

Since we defined green text color for all paragraphs in our stylesheet, you see this text in this color:


p {
  color: green;
}

Class selector

Every HTML element can be assigned with the attribute class. Consequently, we also have a special CSS selector, affecting only HTML elements of specific class. Obviously, such selectors called class selectors.

In our stylesheet we have a class selector called violet, coloring every HTML element of such class into this color. Like this paragraph, belonging to class violet. If you want target elements with a specific class, use a syntax with period (.) character, followed by the class name:


.violet {
  color: violet;
}
This <div> tag also belongs to the violet class.

ID selector

Every HTML element also can have a specific id attribute. It is being used for unique identification of this specific element. The id of the element is unique within a page, so the ID selector is used to select this unique element. This paragraph has an id="unique_paragraph", and it is assigned a crimson text color in pur stylesheet. If you want to style an element with the specific id, write a hash (#) character, followed by the id of the element:


#unique_paragraph {
  color: crimson;
}

Selectors grouping

CSS selectors grouping allows us to combine multiple selectors into a single CSS rule, which could be quite handy. For example, we can assign width of text equal to 300 pixel inside the tag <div> and this particular paragraph, which belongs to class "narrow". If you want to style a group of elements in a same way, just write a comma-separated list of these selectors.


div, .narrow {
  width: 300px;
}

Universal selector

The universal selector (*) unconditionally selects all HTML elements on the page. If you want to create a CSS rule which will affect every HTML element on the page, use the character star (*) before the parameter list. Thus, all elements of this page has Arial font family thanks to universal selector.


* {
  font-family: Arial;
}
