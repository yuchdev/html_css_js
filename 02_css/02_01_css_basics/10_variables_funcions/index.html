Variables

Definition of CSS variable

CSS variable is actually a function var(), that is used to insert a pair key-value into a global scope of the document, or a local scope of some particular selector. CSS variables allow to decrease the necessity of copy&paste of the CSS code, declaring the common values as variables, e.g. same colors.

CSS variable has the following syntax: var(--name, value), where --name is required, and value is optional.
If we define the value, we set it, when we omit it, we actually read previously stored value.
Note: variable name is case-sensitive, and two dashes are required!

In order to create a global variable, use the :root selector. The :root selector matches the document's root. Example:

:root {
  --main-color: #000;
  --main-color: #fff;
}
body {
  background-color: var(--main-color);
}

Redeclaration of variables

If we declare the same variable twice, the second declaration will overwrite the first one. Local variables have higher priority than global variables.

Variables and JavaScript

Variables can be accessed and modified by JavaScript through the DOM model and getComputedStyle() method.
Example:

<script>
  let root = document.documentElement;
  root.style.setProperty('--main-color', '#000');
  console.log(getComputedStyle(root).getPropertyValue('--main-color'));
</script>
