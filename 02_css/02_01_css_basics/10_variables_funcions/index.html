Defenition of CSS variable

CSS variable is actually a function var(), that is used to insert a pair key-value into a global scope of the document, or a local scope of some particular selector. CSS variables allow to decrease the necessity of copy&paste of the CSS code, declaring the common values as variables, e.g. same colors.

CSS variable has the following syntax: var(--name, value), where --name is required, and value is optional.
If we define the value, we set it, when we omit it, we actually read previously stored value.
Note: variable name is case-sensitive, and two dashes are required!
