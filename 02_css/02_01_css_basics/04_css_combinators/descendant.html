Descendant selector example

Here we selects all <p> elements inside <div> elements and apply violet color to them: We make elements more visual by adding different background colors to them and 5 pixels margin to emphasize nesting level, as an inline CSS style.
This text is located within <div> tag. It is not affected, since it is not inside <p> element (has default color).

This text is within the <p> element, which is the first-level descendant of the <div> tag. It is affected by the descendant selector (has violet color)

This text is within the <article> element, which is the first-level descendant of the <div> tag. It is not affected by the descendant selector, since it is not <p> element (has default color).

This text is within the <p> element, which is the first-level of the <article> tag and second-level descendant of the <div> tag. It is affected by the descendant selector (has violet color).
