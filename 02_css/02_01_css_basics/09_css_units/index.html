CSS Units

Definition

Many CSS properties express the size of element. Multiple measure units are supported. Size is being displayed as a number followed by unit without a space between them. Some CSS properties allow negative size.

There are two types of units, relative and absolute.

Absolute Units

Absolute units have fixed length and always appear exactly that size. Possible also to use absolute units are:

  • cm - centimeters
  • mm - millimeters
  • in - inches
  • px - pixels
  • pt - points
  • pc - picas

Note: even pixels are absolute units, they are relative to the viewing device.

Relative Units

Relative units depend on sizing of other units, and they fit better for scaling elements. Possible relative units are:

  • em - relative to the font-size of the element (2em means 2 times the size of the current font)
  • ex - relative to the x-height of the current font (rarely used)
  • ch - relative to the width of the "0" (zero)
  • rem - relative to font-size of the root element
  • vw - relative to 1% of the width of the viewport
  • vh - relative to 1% of the height of the viewport
  • vmin - relative to 1% of viewport's smaller dimension
  • vmax - relative to 1% of viewport's larger dimension
  • % - relative to the parent element
