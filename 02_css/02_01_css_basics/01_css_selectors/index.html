Examples explained

Styling of h1 element

h1 {   color: green;
  text-align: center;

}
  • h1 is a selector in CSS, it points to the HTML element(s) you want to style
  • The declaration block contains one or more declarations separated by semicolons
  • color is a property, and green is the property value
  • text-align is a property, and center is the property value

Styling of div element

div {
  color: maroon;
  margin-left: 100px;
}

Just like <h1> element, all <div> affected elements change their text color to maroon, and become aligned 100 pixels left.

Styling of the single p element

<p style="background-color: violet;">

Here we apply violet background color to a single <p> element.

How to apply CSS rules

There are three ways to apply a stylesheet to an HTML document, and they are applied in exact that order.

  • External
  • Internal
  • Inline

External stylesheet

With the external stylesheet, you can change the look of an entire document or even website by changing just one CSS file. Each HTML page must include a reference to the external style sheet file inside the <link> element, inside the head section. The full example may look like:

<link rel="stylesheet" href="style.css">

An external style sheet is rust a regular text file, saved with a *.css extension. The external *.css file should not contain any HTML tags.

We styled <h1> element, using external CSS stylesheet

Internal stylesheet

An internal style sheet could be used to style a single HTML document. The internal style is defined inside the <style> element, which should be located inside the <head> section. The syntax of CSS code inside of <style> tag is the same with any external *.css file.

We styled <div> element, using external CSS stylesheet

Inline CSS

An inline style may be used to apply a style to a single HTML element. You should just add the style attribute to the relevant element. The style attribute can contain any CSS property. It may look like that:

<h1 style="color:blue;text-align:center;">This is a heading</h1>
