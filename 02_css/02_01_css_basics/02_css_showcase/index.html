As you see, color of 1st-level header is crimson, font size is 32 pixel and is has been put to center. These CSS properties would be applied to all 1st level headers of this document. All paragraphs in the HTML document are colored gray, have font Courier New of size 14 pixel, and the width of every paragraph is 600 pixel.

Meanwhile, 2nd-level heading has not been affected, but it seems to have some specific style already. This style called default and comes from the browser itself. That's why, it may slightly vary from one browser to another.
