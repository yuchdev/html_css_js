Forms in HTML5

Traditionally, the term 'form' has referred to a printed document that contains spaces for you to fill in information. HTML borrows the concept of a form to refer to different elements that allow you to collect information from visitors to your site.

Whether you are adding a simple search box to your website or you need to create more complicated applications, HTML5 forms give you a set of elements to collect data from your users.

  • Adding text
    • Simple text
    • Password
    • Multiline text (text area)
    • Date input
    • Email and URL input
    • Search input
  • Making choices
    • Radio buttons
    • Checkboxes
    • Comboboxes
    • Multiple select boxes
  • Submitting data
    • Submit buttons
    • Image buttons
    • Upload file forms
  • Formatting and validation
    • Group box
    • Labels
    • Basic form validation
