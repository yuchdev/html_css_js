In this example all <h1> elements will be center-aligned, with a green text color

Example explained

h1 {
color: green;
text-align: center;
}
  • h1 is a selector in CSS, it points to the HTML element(s) you want to style
  • The declaration block contains one or more declarations separated by semicolons
  • color is a property, and green is the property value
  • text-align is a property, and center is the property value
