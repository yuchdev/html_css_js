General sibling selector example

Here we select all <p> elements that are next siblings of <div> elements and apply maroon color to them. We make elements more visual by adding different background colors to them and 5 pixels margin to emphasize nesting level, as an inline CSS style.
This text is located within <div> tag. It is not affected, since it is not a sibling <p> element of the <div> element (has default color).

This text is within the <p> element, which comes right next to <div> tag. It is affected by the sibling selector (has maroon color).

This text is within the other <p> element, which is the sibling to a <div> tag. It is affected by the sibling selector (has maroon color).

This text is within the <article> element, which comes next to <div> tag. It is not affected affected by the adjacent sibling selector, since it is not <p> element (has default color).

This text is within the <p> element, which is the first-level descendant of the <article> element. It is not affected by the sibling selector, since it is not a sibling of the <div> tag (has default color).
