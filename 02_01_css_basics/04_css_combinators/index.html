CSS Combinators

The definition of Combinators

The CSS Combinator is a way to define relationships between selectors, to target them more precisely.

Descendant selector

The descendant selector matches all elements that are descendants of a specified element. To create descendant selector put space between such selectors, so that element to the right has been considered a descendant of element to the left. The following example selects all <p> elements inside <div> elements and apply violet color to them:


div p {
  color: violet;
}

Child selector

The child selector selects all elements that are the children (or direct descendants) of a specified element. To create descendant selector put symbol > between such selectors, so that element to the right has been considered a child of element to the left. The following example selects all <p> elements that are children of a <div> element and apply yellow color to them:


div > p {
  color: yellow;
}

General sibling selector

The general sibling selector selects all elements that are next siblings of a specified element. Put symbol ~ so that element to the right has been considered a sibling of element to the left. The following example selects all <p> elements that are next siblings of <div> elements and apply maroon color to them


div ~ p {
  color: maroon;
}

 Link to the example

Adjacent sibling selector

The adjacent sibling selector is used to select an element that is directly after another specific element. Sibling elements must have the same parent element, and "adjacent" is additional requirement to such sibling, meaning "immediately following". The syntax is the same as for general sibling selector, just put + symbol between such elements. The following example selects the first <p> element that are placed immediately after <div> elements and apply blue color to it:


div + p {
  color: blue;
}
