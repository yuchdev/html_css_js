Styling input elements through CSS pseudo-classes

List of related selectors

  • :checked - Select every checked input element
  • :focus - Select every element that has focus
  • :enabled - Select every enabled element
  • :disabled - Select every disabled element
  • :in-range - Select every element that has a value within specific range
  • :out-of-range - Select every element that has a value outside of specific range
  • :valid - Select every element that has a valid value
  • :invalid - Select every element that has an invalid value
  • :required - Select every element that has "required" attribute
  • :optional - Select every element that has no "required" attribute
  • :read-only - Select every element that has "readonly" attribute
  • :read-write - Select every element that has no "readonly" attribute
  • :lang(language) - Select every element with a specific lang attribute value

:enabled and :disabled selectors

The :enabled and :disabled selectors match enabled and disabled elements respectively.

:in-range and :out-of-range selectors

Selectors :in-range and :out-of-range work with values within or outside a specified range respectively. They require input elements to have min and/or max attributes.

Change values within the range of 1-99 and outside it, to see how border style changes.

:valid and :invalid selectors

The :valid and :invalid selectors style elements with a value that validates according to the element's settings. It could be input elements with min and max attributes, email fields with a legal email, or number fields with a numeric value validation.
