Styling links through CSS pseudo-classes

List of related selectors

  • :active - Selects the active link
  • :link - Selects all unvisited link
  • :visited - Selects the visited link
  • :hover - Selects the element on mouse hover
  • :target - Selects the current target element, being linked to URLs followed by #anchor-name

Use the :link selector to style links to unvisited links. A link becomes :active when you click on it. Use the :visited selector to style links to visited pages, and the :hover selector to style links when you mouse over them.

Note 1: :active must come after :hover, and a:hover must come after a:link and a:visited in the CSS definition in order to have effect!

Note 2: in general, you can apply link-related selectors on all elements, not only links, for example, if you want style button to behave like a link.

Link styling and privacy concerns

Browsers limits the styles that can be set for a:visited links, reportedly due to security and privacy concerns. The problem is that :visited selector allows to uncover browser history through JavaScript function window.getComputedStyle(). In order to mitigate this You can apply only following styles:

  • color
  • background-color
  • border-color for all sides
  • outline-color
  • column-rule-color
  • only color of fill and stroke attributes

This is not everything! To preserve user's privacy, Firefox browser will lie to web applications under certain circumstances, always returning values indicating that a user has never visited any of the links on a page.
