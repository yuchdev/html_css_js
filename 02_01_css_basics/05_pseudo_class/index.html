Pseudo-class CSS selectors

Pseudo-class definition

A pseudo-class is a definition of a state of element. For example it could be:

  • Styling a visited and unvisited link differently
  • Styling of an element while mouse hovers over it
  • Styling of an element after it get and lost focus

Pseudo-class syntax and examples

The syntax of pseudo-classes:

selector:pseudo-class {
  property: value;
}

All CSS Pseudo-classes

Link-related Pseudo-classes

  • :active - Selects the active link
  • :link - Selects all unvisited link
  • :visited - Selects the visited link
  • :hover - Selects the element on mouse hover
  • :target - Selects the current active element (e.g. clicked on a URL containing that anchor name)

See also styling links through pseudo-classes.

Input and language-related Pseudo-classes

  • :checked - Select every checked element
  • :focus - Select every element that has focus
  • :enabled - Select every enabled element
  • :disabled - Select every disabled element
  • :in-range - Select every element that has a value within specific range
  • :out-of-range - Select every element that has a value outside of specific range
  • :valid - Select every element that has a valid value
  • :invalid - Select every element that has an invalid value
  • :required - Select every element that has "required" attribute
  • :optional - Select every element that has no "required" attribute
  • :read-only - Select every element that has "readonly" attribute
  • :read-write - Select every element that has no "readonly" attribute
  • :lang(language) - Select every element with a lang attribute value starting with "language"

Pseudo-classes based on relationships between elements

  • :empty - Select every element that has no children
  • :only-child - Selects every element that is the only child of this type of its parent
  • :only-of-type - Selects every element that is the only element of this type of its parent
  • :root - Selects the document's root element
  • :not(selector) - Select every element that is not of selector type
  • :first-child - Select every element that is the first child of its parent
  • :last-child - Select every element that is the last child of its parent
  • :nth-child(n) - Select every element that is the n-th child of its parent
  • :nth-last-child(n) - Select every element that is the n-th from the end child of its parent
  • :first-of-type - Select every element that is the first of this type of its parent
  • :last-of-type - Select every element that is the last of this type of its parent
  • :nth-of-type(n) - Select every element that is the n-th element of this type of its parent
  • :nth-last-of-type(n) - Select every element that is the n-th element from the end of this type of its parent
