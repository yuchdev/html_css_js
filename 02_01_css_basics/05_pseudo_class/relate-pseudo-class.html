Styling elements with different relationships through pseudo-classes

We make elements more visual by adding different background colors to them as an inline CSS style.

List of related selectors

:root selector

The :root selector matches the while document's root element.

:empty selector

The :empty selector matches every element that has no children (including text content).

N-th child pseudo-classes

  • :first-child - Select every element that is the first child of its parent
  • :last-child - Select every element that is the last child of its parent
  • :nth-child(n) - Select every element that is the n-th child of its parent
  • :nth-last-child(n) - Select every element that is the n-th from the end child of its parent
The :nth-child(n) selector used to style element that is the N-th child, regardless of type, of its parent. N can be a number, a keyword (live odd), or a formula (like 3n). Could be handy to style every 2nd table or list row for better visualization.

:only-child and :only-of-type selector

N-th of type pseudo-classes

  • :first-of-type - Select every element that is the first of this type of its parent
  • :last-of-type - Select every element that is the last of this type of its parent
  • :nth-of-type(n) - Select every element that is the n-th element of this type of its parent
  • :nth-last-of-type(n) - Select every element that is the n-th element from the end of this type of its parent
The :nth-of-type(n) selector matches every element that is the nth child, of a particular type. N also can be a keyword or formula.
