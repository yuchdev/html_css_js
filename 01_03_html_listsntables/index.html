Lists

Notes about lists

  • Each item in the list is placed between an opening <li> tag and a closing </li> tag. (The li stands for list item.)
  • Lists can contain only <li> elements, no text
  • Sometimes you may see a type attribute used with the <ol> of <ul> element to specify the type of numbering (numbers, letters, roman numerals and so on) or bullets (circles, squares, diamonds and so on)
    It is better to use the CSS liststyle-type property.

Unordered lists

Browsers indent lists inordered by default. Sometimes you may see a type attribute used with the <ul> element to specify the type of bullet point (circles, squares, diamonds and so on).
It is better to use the CSS liststyle-type property.

Ordered lists

Browsers indent lists ordered by default. Sometimes you may see a type attribute used with the <ol> element to specify the type of numbering (numbers, letters, roman numerals and so on).
It is better to use the CSS liststyle-type property.

Description lists

The defnition list is created with the <dl> element and usually consists of a series of terms and their defnitions.
Inside the <dl> element you will usually see pairs of <dt> and <dd> elements.
Inside a <dd> tag you can put paragraphs, line breaks, images, links, lists, etc.
Sometimes you might see a list where there are two terms used for the same defnition or two different defnitions for the same term.

Sashimi
Sliced raw fish that is served with condiments such as shredded daikon radish or ginger root, wasabi and soy sauce.

Nested lists

You can put a second list inside an <li> element to create a sublist or nested list. Browsers display nested lists indented further than the parent list.
In nested unordered lists, the browser will usually changethe style of the bullet point too.
In ordered, you should use CSS to style numerals.

Tables

Table basics

The <table> element is used to create a table. The contents of the table are written out row by row.
To indicate the start of each row using the opening <tr> tag. The <tr> stands for table row.
Each cell of a table is represented using a <td> element. The <td> stands for table data.
the <td> element defines a table header.
A more complex HTML table may also include <caption>, <col>, <colgroup>, <thead>, <tfoot>, and <tbody> elements.
There are a number of table attributes, like "align", "bgcolor", "border", "cellpadding", "cellspacing", "frame", "rules", "summary", and "width". These attributes are obsolete and not supported in HTML5.
Use CSS styling instead
