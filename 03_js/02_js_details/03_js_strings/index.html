JavaScript Strings

Definition of a String

  • A string in JavaScript is a sequence of characters
  • A string is a primitive data type, so creating string object using String() constructor is not necessary
  • A string is a sequence of characters that can be used to represent text
  • Strings are immutable, meaning that they cannot be changed once created
  • Strings are created by enclosing characters in single or double quotes

Use of quotes

Both single- and double-quoted strings are valid in JavaScript. However, there are some differences between them:

  • Double-quoted strings can contain single quotes
  • Single-quoted strings can contain double quotes
  • Double-quoted strings can contain escape characters
  • Single-quoted strings cannot contain escape characters

Escape characters are used to represent characters that cannot be typed directly into a string. For example, the \n escape character represents a new line, or double-quote inside a double-quoted string: "\"".

Other escape characters valid in JavaScript:

  • \n - new line
  • \t - tab
  • \b - backspace
  • \r - carriage return
  • \f - form feed
  • \v - vertical tab
  • \\ - backslash
  • \' - single quote
  • \" - double quote
  • \xnn - hexadecimal character
  • \unnnn - Unicode character

String can be broken into multiple lines by using the \ escape character at the end of each line.

String Methods

JavaScript provides a number of methods for manipulating strings. These methods are called on a string object, and they return a new string object. The original string is not changed.

JavaScript string offers several types of methods:

  • String length: length
  • Searching for a string: search(), match(), includes(), startsWith() and others
  • Extracting string parts: slice(), substring(), substr(), split()
  • Replacing string content: replace(), replaceAll()
  • Converting to upper and lower case: toUpperCase(), toLowerCase()
  • Converting a string to an array: split()
  • Converting a string to a number: Number(), parseInt(), parseFloat()
  • Converting a number to a string: toString()
  • String padding: padStart(), padEnd()

Since string is immutable, all string methods return a new string. Accessor methods like operator [] and charAt() do not change character in the original string, but if you attempt to change a character in a string, JavaScript will not throw an error: 

    
        let str = "Hello";
        str[0] = "M"; // No error
        console.log(str); // Hello

HTML Wrapper Methods

JavaScript provides a number of non-standard methods for returning HTML-wrapped strings, e.g. anchor() for creating an anchor, big() for creating a big text, bold() for creating a bold text, etc.

Template Literals

Template literals are string literals allowing embedded expressions. You can use multi-line strings and string interpolation features with them. They were called "template strings" in prior editions of the ES2015 specification.

  • You can include both single- and double-quoted strings in template literals: `"Hello JS"`
  • You can use escape characters in template literals: `Hello \t JS`
  • You can use multi-line strings in template literals
  • You can substitute variables and expressions in template literals: `Hello ${name}`

Regular Expressions in JavaScript

JavaScripts supports regular expressions, which are patterns used to match character combinations in strings.
