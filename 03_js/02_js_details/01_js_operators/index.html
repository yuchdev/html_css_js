JavaScript Operators

Arithmetic Operators

Basic arithmetic in JS is intuitive and follows the same rules as in math.
Operators precedence also follows the same rules as in math.
Available operators are: +, -, *, /, %, ++, --, **, where ** is exponentiation, and % is the remainder of the division.

Arithmetic Operators & Assignment

All arithmetic operations can be combined with assignment operators, e.g. +=, -=, *=, /=, %=, **=.

Operations with strings

Operation + works as concatenation for strings, e.g. "a" + "b" = "ab". When a string is added to a number, the result is a string concatenation, e.g. "1" + 2 = "12"

Comparison Operators

Comparison operators are used to compare two values, and return a boolean value. Available operators are: ==, ===, !=, !==, >, <, >=, <=.
Note the difference between == and === operators, as well as != and !== operators. The latter is strict equality, including type of compared values.

Comparison with null, undefined and NaN

null and undefined are equal to each other, but not to any other value. NaN is result of incorrect mathematical operations, and is not equal to anything, and is not equal to anything, including itself.

Logical Operators

Logical operators are used to determine the logic between variables or values. Available operators are: &&, ||, !.

Nullish Coalescing Operator

The nullish coalescing operator ?? returns the first operand if it is not null or undefined, otherwise the second operand, e.g.

let a = 1;
let b = 0;
let c = a ?? b; // c = 1
let d = null ?? b; // d = 0
let e = undefined ?? b; // e = 0
let f = 0 ?? b; // f = 0

Optional Chaining Operator

The optional chaining operator ?. allows to access properties of nested objects without checking if the object exists, e.g.

let user = {
name: "John",
age: 30,
address: {
street: "Main",
number: 1
}
};
let street = user.address.street; // street = "Main"
let number = user.address.number; // number = 1
let city = user.address.city; // city = undefined
let city = user?.address?.city; // city = undefined

Bitwise Operators

Bitwise operators treat their operands as a set of 32 bits (zeros and ones). Bitwise operators perform their operations on such binary representations, and return standard 64-bits JavaScript number.

JS offers standard set of bitwise operators: &, |, ^, ~, <<, >>, >>>.
Note the difference between >> and >>> operators.
The first one is signed right shift, which pushing leftmost bit to the right, and the rightmost bits fall off. The second one is unsigned right shift, which pushing zeros from the left, and the rightmost bits fall off.
