HTML DOM Model

HTML DOM Tree

The HTML DOM is a standard for how to get, change, add, or delete HTML elements. Through the HTML DOM, JavaScript can access and change all the elements of an HTML document:

  • Change all the HTML elements in the page
  • Change all the HTML attributes in the page
  • Change all the CSS styles in the page
  • React to all new and existing HTML events in the page
  • Create and delete HTML elements and attributes

HTML DOM Methods and Properties

The HTML DOM is a standard object model and programming interface for HTML. It defines:

  • The methods (or actions) you can perform on HTML elements
  • The events for all HTML elements
  • The properties (or values) of all HTML elements

Finding HTML Elements

You can access any HTML element on the page, but first you have to find it. These are the most common methods for finding HTML elements:

  • document.getElementById(id)
  • document.getElementsByTagName(name)
  • document.getElementsByClassName(name)
  • document.querySelector(cssSelector)
  • document.querySelectorAll(cssSelector)

Changing HTML Elements

After you have found the HTML element you want to change, you can change the element content, attributes, and CSS styles. There are properties and methods to do so:

  • innerHTML - Sets or returns the HTML content (inner HTML) of an element
  • attribute - Sets or returns the value of the specified attribute
  • style.property - Sets or returns the value of an element's style property
  • setAttribute(attribute, value) - Adds a new attribute, or changes the value of an existing attribute
  • removeAttribute(attribute) - Removes an attribute

Adding and Deleting HTML Elements

You can add new HTML elements to the page, or remove existing HTML elements from the page. These are the methods for adding and deleting HTML elements:

  • document.createElement(element) - Creates an HTML element
  • document.removeChild(element) - Removes an HTML element
  • document.appendChild(element) - Adds an HTML element
  • document.replaceChild(new, old) - Replaces an HTML element
  • document.write(text) - Writes into the HTML output stream

Adding and Removing HTML Event Handlers

You can add event handlers to HTML elements, to make them react to user input. These are the methods for adding and removing event handlers:

  • document.getElementById(id).onclick = function(){code} - Adds an event handler to an HTML element
  • element.addEventListener(event, function, useCapture) - Adds an event handler to the specified element
  • element.removeEventListener(event, function, useCapture) - Removes an event handler from the specified element
