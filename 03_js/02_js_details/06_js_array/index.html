JavaScript Arrays

Definition

Just like in other languages, array is a special variable, which can hold more than one value at a time. Using of arrays in JavaScript is intuitive:

let cars = ["Saab", "Volvo", "BMW"];

Also, you can create an empty array and then add elements to it. Operator [] is used to access array elements.

    let cars = [];
    cars[0] = "Saab";
    cars[1] = "Volvo";
    cars[2] = "BMW";

Array is an object, so it can hold properties and methods. It is a list-like object whose prototype has methods to perform traversal and mutation operations. Array elements can be primitive types, objects, functions and other arrays.

NOTE: It is a common practice to declare array const. However, it declares const reference to the array, rather than the array itself. So, it is possible to change array elements, but not to reassign the array.

Array Properties

Array.length

The length property of an object which is an instance of type Array sets or returns the number of elements in that array. The value is an unsigned, 32-bit integer that is always numerically greater than the highest index in the array.

Array.prototype

The prototype property allows you to add new properties and methods to an Array object:

  Array.prototype.myUcase = function() {
    for (i = 0; i < this.length; i++) {
      this[i] = this[i].toUpperCase();
    }
  };

Array.toString()

The toString() method returns a string representing the specified array and its elements:

  let fruits = ["Banana", "Orange", "Apple", "Mango"];
  let energy = fruits.toString(); // Banana,Orange,Apple,Mango

Method join() is similar to toString() but it allows you to specify the separator:

  let fruits = ["Banana", "Orange", "Apple", "Mango"];
  let energy = fruits.join(" * "); // Banana * Orange * Apple * Mango

Array Iteration

The forEach() method executes a provided function once for each array element, e.g.

  let cars = ["Saab", "Volvo", "BMW"];   cars.forEach(function(item, index, array) {     console.log(item, index);   });

Push and Pop

JavaScript arrays offer push() and pop() methods to add and remove elements from the end of an array, so as shift() and unshift() to add and remove elements from the beginning of an array, e.g.

  let cars = ["Saab", "Volvo", "BMW"];
  cars.push("Audi");
  cars.pop();
  cars.shift();
  cars.unshift("Audi");
  console.log(cars); // Audi,Volvo,BMW

Deleting Elements

The delete operator removes the specified property or array element from the object. It returns true if the operation succeeds, otherwise it returns false. The delete operator does not affect arrays with numbered indexes. Be aware, it leaves undefined holes in the array. Try to use splice(), pop() or shift() instead.

Splice and Slice

The splice() method adds/removes items to/from an array, and returns the removed item(s). The first parameter (2) defines the position where new elements should be added (spliced in). The second parameter (0) defines how many elements should be removed. The rest of the parameters ("Lemon" , "Kiwi") define the new elements to be added. The slice() method returns the selected elements in an array, as a new array object. The slice() method selects the elements starting at the given start argument, and ends at, but does not include, the given end argument, e.g.

  let fruits = ["Banana", "Orange", "Lemon", "Apple", "Mango"];
  fruits.splice(2, 0, "Lemon", "Kiwi");
  console.log(fruits); // Banana,Orange,Lemon,Kiwi,Apple,Mango
  let citrus = fruits.slice(1);
  console.log(citrus); // Orange,Lemon,Kiwi,Apple,Mango

Array Sorting

The sort() method sorts the items of an array. The sort order can be either alphabetic or numeric, and either ascending (up) or descending (down). By default, the sort() method sorts the values as strings in alphabetic and ascending order. This works well for strings ("Apple" comes before "Banana"). However, if numbers are sorted as strings, "25" is bigger than "100", because "2" is bigger than "1". Because of this, the sort() method will produce an incorrect result when sorting numbers. You can fix this by providing a "compare function". The compare function should return a negative, zero, or positive value, depending on the arguments, like this:

  let points = [40, 100, 1, 5, 25, 10];
  points.sort(function(a, b){return a - b});
  console.log(points); // 1,5,10,25,40,100

NOTE: keeping all array elements the same type is a good practice.

There's no built-in methods to find the maximum or minimum of an array of numbers. Instead, you can use the Math.max() and Math.min() functions, e.g.

  let points = [40, 100, 1, 5, 25, 10];
  let max = Math.max.apply(null, points);
  let min = Math.min.apply(null, points);
  console.log(max); // 100
  console.log(min); // 1

If the array contains objects, you can use the sort() method to sort the array by a specific object property. The following example sorts an array of objects by the price property:

  let cars = [
    {type:"Volvo", price:50000},
    {type:"Saab", price:20000},
    {type:"BMW", price:30000}
  ];
  cars.sort(function(a, b){return a.price - b.price});
  console.log(cars); // Saab,BMW,Volvo

Useful Array Methods

Map, Filter and Reduce

The map() method creates a new array with the results of calling a function for every array element. The map() method creates a new array with the results of calling a function for every array element. The map() method calls the provided function once for each element in an array, in order. Note: map() does not execute the function for array elements without values. Note: map() does not change the original array. Example:

  let numbers = [4, 9, 16, 25];
  let x = numbers.map(Math.sqrt);
  console.log(x); // 2,3,4,5

The filter() method creates a new array with array elements that passes a test. The filter() method creates a new array with array elements that passes a test. The filter() method checks if all array values pass a test. Example:

  let numbers = [4, 9, 16, 25];
  let x = numbers.filter(function(value, index, arr){
    return value > 15;
  });
  console.log(x); // 16,25

The reduce() method reduces the array to a single value. The reduce() method runs a function on each array element to produce (reduce it to) a single value. The reduce() method works from left-to-right in the array. There's version of the reduce() method that works from right-to-left, called reduceRight(). Example:

  let numbers = [4, 9, 16, 25];
  let x = numbers.reduce(function(total, value, index, arr){
    return total + value;
  });
  console.log(x); // 54

Every and Some

The every() method checks if all array values pass a test, and the some() method checks if some array values pass a test, e.g.

  let numbers = [4, 9, 16, 25];
  let x = numbers.every(function(value, index, arr){
    return value > 15;
  });
  console.log(x); // false
  let y = numbers.some(function(value, index, arr){
    return value > 15;
  });
  console.log(y); // true

Search in Array

The indexOf() method searches an array for a specified item and returns its position. The indexOf() method searches an array for a specified item and returns its position. If the item is not found, it returns -1. The indexOf() method is case sensitive. Example:

  let fruits = ["Banana", "Orange", "Apple", "Mango"];
  let a = fruits.indexOf("Apple");
  console.log(a); // 2

The lastIndexOf() method searches an array for a specified item, and returns its position. The lastIndexOf() method searches an array for a specified item, and returns its position. The search will start at the specified position, or at the end if no start position is specified, and end the search at the first occurrence. If the item is not found, it returns -1. The lastIndexOf() method is case sensitive. Example:

  let fruits = ["Banana", "Orange", "Apple", "Mango"];
  let a = fruits.lastIndexOf("Apple");
  console.log(a); // 2

The find() method returns the value of the first array element that passes a test function. The find() method returns the value of the first array element that passes a test function. If no values pass the test, undefined is returned. Example:

  let numbers = [4, 9, 16, 25, 29];
  let first = numbers.find(function(value, index, arr){
    return value > 18;
  });
  console.log(first); // 25

The findIndex() method returns the index of the first array element that passes a test function, and lastIndexOf() returns the index of the last array element that passes a test function.

The includes() method checks if an array contains the specified element. The find() method returns the value of the first array element that passes a test function. Example:

  let fruits = ["Banana", "Orange", "Apple", "Mango"];
  let a = fruits.includes("Mango");
  let b = fruits.find("Kiwi");
  console.log(a); // true
  console.log(b); // undefined

JavaScript Iterables

JS Iterables is a new feature in ECMAScript 6. It is an object that provides a way to iterate over arrays and other collections:

  let numbers = [4, 9, 16, 25];
  let x = numbers.entries();
  for (let i of x) {
    console.log(i);
  }
// [0, 4], [1, 9], [2, 16], [3, 25]

We can iterate over the keys, values, including sets and maps using the keys(), values(), and entries() methods, e.g.

  let numbers = [4, 9, 16, 25];
  let x = numbers.keys();
  for (let i of x) {
    console.log(i);
  }
// 0, 1, 2, 3
  let y = numbers.values();
  for (let i of y) {
    console.log(i);
  }
// 4, 9, 16, 25
  let z = numbers.entries();
  for (let i of z) {
    console.log(i);
  }
// [0, 4], [1, 9], [2, 16], [3, 25]
