Ways to display text in JavaScript

JavaScript does not have access to any system output functions and devices. The only way to show debugging information is accessing document itself, or browser debugging tools.
JavaScript can "display" data in 4 ways:

  • Writing into an HTML element, using document.getElementById(element).innerHTML
  • Writing into the HTML output using document.write()
  • Showing to the user an alert box, using window.alert()
  • Writing into the browser console, using console.log()

Writing into the HTML output using document.write()

The document.write() method writes into the HTML output stream. It overwrites the whole HTML document, use it only for testing (reload the page to restore the original document).

Showing to the user an alert box

The window.alert() method displays an alert box with a message and an OK button. It can be used to display an important message, usually warnings or errors.

Writing into the browser console

And at last, the console.log() method writes into the browser console. Open the console first to see the output, usually it is located in developer tools (displayed by pressing F12).
