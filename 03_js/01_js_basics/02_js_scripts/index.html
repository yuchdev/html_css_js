JavaScript Script Location

Locations of JavaScript Scripts

JavaScript can be placed in the HTML document in three different places:

  • In the <head> section
  • Inside the HTML document (inside the <body> section)
  • In an external file, and referenced in the HTML document

JavaScript in the <head> section

We located inside the <head> the following script, detecting the name of the HTML document, and displaying it in the next paragraph:

<script> function showDocumentName() {
  let docName = document.URL;
  docName = docName.substring(docName.lastIndexOf('/') + 1);
  docName = docName.substring(0, docName.lastIndexOf('?'));
  document.getElementById("doc_name").innerHTML = "The script is name is " + docName;
} </script>

The script is name is ...

JavaScript inside the HTML document

We can place this script changing the paragraph style in the <body> section:

  <script>
  document.getElementById("style_change").style.color = "red";
  styleChange.style.backgroundColor = "yellow";
  </script>

JavaScript in an external file

We can place the script changing this paragraph style in an external file, and reference it in the HTML document header:

<script src="external.js"></script>

The script called external.js, and contains the following code:

  function changeStyleExternal() {
    let styleChange = document.getElementById("style-change-2");
    p.style.color = "red";
    p.style.fontFamily = "Arial";
    p.style.fontSize = "16px";
  }

Locating JS external file is the best practice, as it gives developer several advantages:

  • It separates HTML and code
  • It makes HTML and JavaScript easier to read and maintain
  • Cached JavaScript files can speed up page loads

Locating JS script in the Web

The src attribute could be used to locate the script in the Web. The src attribute can contain the full URL of the script, or a relative URL:

<script src="https://www.w3schools.com/js/myScript1.js"></script>
<script src="https://www.w3schools.com/js/myScript2.js"></script>
<script src="/js/myScript3.js"></script>
<script src="myScript4.js"></script>
