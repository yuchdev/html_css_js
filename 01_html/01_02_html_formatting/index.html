Demonstration of formatting tags

Basic Formatting

Bold text <b>

Tag <b>: bold text, defines text with strong importance. According to the HTML5 specification, the <b> tag should be used as a last resort when no other tag is more appropriate, use <mark>, <strong> and <em> instead.
Tag <b> is identical to CSS font-weight: bold

Italic text <i>

Tag <i>: italic text, the element defines a part of text in an alternate voice or mood.
Often used to indicate a technical term, a phrase from another language, a thought, or a feeling.
Just like <b>, <i> should be used when no other tag is more appropriate. Check out <mark>, <strong>, <em>, <cite>, and <dfn> instead.
Tag <i> is identical to CSS font-weight: italic

Subscript <sub> and Superscript <sup>

Tags <sub> and <sup> tags define subscript and superscript text respectively, which appear half a character below or above the normal text line, and rendered with smaller font.

Subscript can be used for chemical formulas, for example:
The amount of CO2 in the atmosphere grew by 2ppm in 2009.

Superscript text can be used for footnotes or raise to power mathematical expressions:
The number 10100 is called Googolplex.

Semantic Highlight

Emphasize tag <em>

The <em> tag is used to define emphasized text, which is meant to be pronounced with stressed intonation, and subtly changes the meaning of a sentence.
The content inside is typically displayed in italic.

Important text <strong>

The <strong> tag is used to define important text. The content inside is typically displayed in bold. The tag <b> on the other hand, is used to specify bold text without any additional importance.

Smaller text <small>

The <small> tag is used to define smaller text, like copyright and other side-comments. If the smaller text does not have special semantic meaning, use CSS for formatting instead.

Highlighted text <mark>

The <mark> tag is used to define text that should be highlighted . Most browsers will display the such element with the yellow background color.

Incorrect text <s>

The <s> tag is used to define text that is no longer correct, accurate or relevant. The text will be displayed with a line through it. The <s> tag should not be confused with <del>, which used to define deleted text in a document.
Tag <s> is identical to CSS text-decoration: line-through

Underlined text <u>

The <u> tag is used to define underlined text. Could be used for variety of purposes, like misspelled words.
Tag <u> is identical to CSS text-decoration: underline
Tip: Avoid using the <u> element where it could be confused with a hyperlink <a>.

Formatting Text

Possible Line Break <wbr>

When a word is too long, the browser might break it at the wrong place. You can use the <wbr> element to add word break opportunities.

Pre-Formatted Text <pre>

        Text in a <pre> element is displayed in a fixed-width font (usually Courier),
        and it preserves both spaces and line breaks.    (4 spaces)
        The text will be displayed exactly as written in the HTML source code.

Hidden Content <template>

The <template> tag is used as a container to hold some HTML content hidden from the user when the page loads. Such content can be rendered later with a JavaScript. You can use it for delayed display of the content or as a placeholder for a dynamically loaded content. TODO: example!

Date and Time

The <time> tag defines a human-readable date/time (see source for examples):
We open at 10:00 every morning
I have a date on Valentines day

The datetime attribute of the <time> element is used to translate the time into a machine-readable format, so that it could be processed by other applications (browser itself, calendar, search engine), e.g.
<time datetime="2019-02-14 20:00">

Meter <meter> or progress bar <progress>

The <meter> tag defines a scalar measurement within a known range, or a fractional value.
Examples: Disk usage, the relevance of a query result, etc.

Display a meter:


2 out of 10 <meter value="2" min="0" max="10">2 out of 10</meter>
60% <meter value="0.6">60%</meter>

Note! The <meter> tag should not be used to indicate dynamic progress (as in a progress bar). For progress bars, use the <progress> tag:
<progress value="22" max="100"></progress>

Quotation

Blockquotes <blockquote>

The <blockquote> element is used for longer quotes that take up an entire paragraph. Note how the <p> element is still used inside the <blockquote> element.

Browsers tend to indent the contents of the <blockquote> element, however you should not use this element just to indent a piece of text, rather you should achieve this effect using CSS.

HTML and CSS, Jon Ducket

Quotations <q>

The <q> element is used for shorter quotes that sit within a paragraph. Browsers are supposed to put quotes around the <q> element, however Internet Explorer does not — therefore many people avoid using the <q> element. Both elements may use the cite attribute to indicate where the quote is from (HTML and CSS, Jon Ducket) Its value should be a URL that will have more information about the source of the quotation.

Abbreviations, Acronyms and References

The title of work <cite>

When you are referencing a piece of work such as a book (poem, song, movie, painting, sculpture, etc.), the <cite> element can be used to indicate the title of such work. Note, the author's name is not a title of work.
The text in the <cite> element usually renders in italic.
Example:

A Brief History of Time by Stephen Hawking has sold over ten million copies worldwide

Definition <dfn>

The first time you explain some new terminology (perhaps an academic concept or some jargon) in a document, it is known as the defining instance of it. The <dfn> element is used to indicate the defining instance of a new term. A title attribute on the opening <dfn> tag is used to specify the full term, which is displayed when you mouse over the element.
The nearest parent of the <dfn> must also contain the definition/explanation for the term. Some browsers show the content of the <dfn> element in italics. Safari and Chrome do not change its appearance.

Abbreviation or Acronym <abbr>

The <abbr> tag defines an abbreviation or an acronym, e.g. HTML, CSS, Dr., ASAP

Use of <abbr title="Professor"> tag:
Prof. Stephen Hawking is a theoretical physicist and cosmologist.
If you use an abbreviation or an acronym, then the element can be used. A title attribute on the opening <abbr> tag is used to specify the full term, which is displayed when you mouse over the element.
Abbreviation can be included inside the <dfn> element.

Information about the author <address>

The <address> element has quite a specific use: to contain details about the author of the page, document or article.
It can contain a physical address, but it does not have to. For example, it may also contain a phone number or email address. Browsers often display the content of the <address> element in italics, and browsers add a line break before and after the content of the <address> element.
Tag <address> is not allowed within tag <p>
Example:

homer@example.org

742 Evergreen Terrace, Springfield.
