HTML Events

Concept

HTML lets you trigger action in browser, e.g. when you click on a button, when you press a key, when you scroll the page, etc. The action could be anything that HTML/CSS/JS allows, e.g. a page reload, a form submission, window resize, style change, running script, etc.

Some events may occur in any HTML element starting from the document root, e.g. when you reload page, when you resize window, etc. Some events are specific to elements, good example are forms: when you submit a form, when you change a form field value, obtain or lose focus, etc.

Global event examples

  • Page reload: <body onload="alert('Hello World')">
  • Window resize: <body onresize="alert('Window resized')">
  • Windows closed: <body onunload="alert('Window closed')">

Form-specific event examples

  • Form submitted: <form onsubmit="alert('Form submitted')">
  • Form field value changed: <input onchange="alert('Field value changed')">
  • Form field gained focus: <input onfocus="alert('Field gained focus')">
  • Form field lost focus: <input onblur="alert('Field lost focus')">

Keyboard and mouse event examples

  • Key pressed: <body onkeydown="alert('Key pressed')">
  • Key released: <body onkeyup="alert('Key released')">
  • Mouse clicked: <body onclick="alert('Mouse clicked')">
  • Mouse double-clicked: <body ondblclick="alert('Mouse double-clicked')">
  • Mouse pressed: <body onmousedown="alert('Mouse button pressed')">
  • Mouse released: <body onmouseup="alert('Mouse button released')">
  • Mouse moved: <body onmousemove="alert('Mouse moved')">
  • Mouse left: <body onmouseleave="alert('Mouse left button')">
  • Mouse wheel: <body onwheel="alert('Mouse wheel')">

Other event examples: clipboard, drag-n-drop, media

  • Clipboard: <body oncopy="alert('Copy to clipboard')">
  • Drag-n-drop: <body ondrag="alert('Drag started')">
  • Media: <body oncanplay="alert('File start playing')">
